## SQL: Fundamentos

### Introdução

Structured Query Language (SQL) é uma linguagem de programação especializada para gerenciar dados em sistemas de banco de dados relacionais (RDBMS), podendo ser aplicada em Sistemas de Gerenciamento de Dados em Fluxo Relacional (RDSMS) ou bancos de dados NoSQL.

### Sublinguagens do SQL

SQL é dividido em três sublinguagens principais:

#### 1. Data Definition Language (DDL)

- **Propósito:** Criar e modificar a estrutura do banco de dados.
- **Exemplos:** CREATE, ALTER, DROP.

#### 2. Data Manipulation Language (DML)

- **Propósito:** Executar operações CRUD nos dados do banco.
- **Comandos:** INSERT, SELECT, UPDATE, DELETE.

#### 3. Data Control Language (DCL)

- **Propósito:** Controlar o acesso aos dados no banco.
- **Exemplos:** GRANT, REVOKE.

[Leia mais na Wikipedia](https://en.wikipedia.org/wiki/SQL)

### Operações CRUD

- **CREATE:** Cria novos elementos (tabelas, índices).
- **READ (SELECT):** Recupera dados.
- **UPDATE:** Modifica dados existentes.
- **DELETE:** Remove linhas.

- **MERGE:** Realiza operações de escrita combinadas (INSERT, UPDATE, DELETE).
